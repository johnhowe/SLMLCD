Description
sdmmc_spi.h::sdmmc_spi
 
Purpose
Implementation for sdcard spi mode physical layer driver. Supply a set of sdcard spi mode's interface.

Usage
  1. SD_Init: Run the SDcard initialization sequence
  2. SD_SPI_Init : Run the SDcard SPI Mode initialization sequence
  3. SD_Stop: Stop the SDcard by sending Cmd12
  4. SD_ReadBlock : Read blocks of data
  5. SD_WriteBlock : Write blocks of data
