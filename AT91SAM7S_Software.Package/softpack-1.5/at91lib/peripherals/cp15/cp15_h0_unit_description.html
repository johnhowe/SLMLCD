Description
Purpose
Methods to manage the Coprocessor 15. Coprocessor 15, or System Control Coprocessor CP15, is used to configure and control all the items in the list below: • ARM core • Caches (ICache, DCache and write buffer) • TCM • MMU • Other system options

Usage
  1. Enable or disable D cache with Enable_D_Cache and Disable_D_Cache
  2. Enable or disable I cache with Enable_I_Cache and Disable_I_Cache
