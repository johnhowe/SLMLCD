Purpose
This directory contains structures, definitions and functions for AT91 UDP. They can be divided into two groups:
  • Hardware layer: low-level operations on the USB UDP controller
    • USBD_OTGHS.c USBD_UDP.c USBD_UDPHS.c
  • USB Device API: other files in the directory, offers hardware-independent methods and structures
