Description
Purpose
Definitions of the descriptors required by the HID device keyboard driver.

Usage
  1. Use the hiddMouseDriverDescriptors variable to initialize a USBDDriver instance.
  2. Send hiddReportDescriptor to the host when a GET_DESCRIPTOR request for the report descriptor is received.
