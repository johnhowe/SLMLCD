Description
Purpose
Definition of a class for manipulating HID keyboard output reports.

Usage
  1. Initialize a newly-created output report instance with HIDDKeyboardOutputReport_Initialize.
  2. Retrieve the status of the three LEDs using HIDDKeyboardOutputReport_GetNumLockStatus, HIDDKeyboardOutputReport_GetCapsLockStatus and HIDDKeyboardOutputReport_GetScrollLockStatus.
