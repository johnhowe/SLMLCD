Description
Purpose
Definitions of constants and methods for the HID keypad usage page.

Usage
  1. Use the constants declared in this file when instanciating a Report descriptor instance.
  2. When implementing the functionality of an HID keyboard, use the key codes defined here to indicate keys that are being pressed and released.
