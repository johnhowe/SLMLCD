Description
Purpose
Definitions and methods for USB configuration descriptor structures described by the USB specification.

Usage
  1. Declare USBConfigurationDescriptor instance as a part of the configuration descriptors of a USB device.
  2. To get useful information (field values) from the defined USB device configuration descriptor, use
    • USBConfigurationDescriptor_GetTotalLength
    • USBConfigurationDescriptor_GetNumInterfaces
    • USBConfigurationDescriptor_IsSelfPowered
  3. To pase the defined USB device configuration descriptor, use
    • USBConfigurationDescriptor_Parse
