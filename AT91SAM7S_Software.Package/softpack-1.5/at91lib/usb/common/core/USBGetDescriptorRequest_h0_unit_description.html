Purpose
Definition of the USBGetDescriptorRequest class.

Usage
  • After a GET_DESCRIPTOR request has been received, retrive the useful values with following functions:
    • USBGetDescriptorRequest_GetDescriptorType: the descriptor type
    • USBGetDescriptorRequest_GetDescriptorIndex: the index of the requested descriptor
